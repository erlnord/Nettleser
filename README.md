# **Volant** by Team OS
Det her er en nettleser. Snart iallefall. 

Målet med dette prosjektet er å lage en sikker og brukervennlig nettleser til Android. Prosjektet har fokus på personvern og sikkerhet.

#### Features
* Reverse image search
* Bokmerker
* Tabs
